# UNIT-PSYCH — Cognitive & Motivational Bias Catalog (80+)

Reference for UNIT-PSYCH (Behavioral Forensic Sub-Agent). Operative list lives in
the sub-agent system_prompt (`orchestrator/sub_agent_spawner.py`); this file is the
annotated human reference. English-only per skills/domain convention.

## 1. Belief / Confirmation
- Confirmation bias — seeks/weights evidence that confirms priors.
- Congruence bias — tests only the hypothesis held, not alternatives.
- Myside bias — evaluates evidence favorably to one's own position.
- Belief perseverance — retains beliefs after disconfirmation.
- Motivated reasoning — conclusion driven by desired outcome.
- Semmelweis reflex — rejects evidence contradicting a paradigm.
- Backfire effect — counter-evidence strengthens the original belief.
- Cherry-picking — cites supporting data, omits contradicting.
- Selective perception — expectations shape what is noticed.
- Observer-expectancy — researcher expectation biases results.

## 2. Optimism / Overconfidence
- Optimism bias — underestimates likelihood of negative outcomes.
- Overconfidence effect — excessive certainty in own judgment.
- Planning fallacy — underestimates time/cost/risk of plans.
- Dunning-Kruger — low competence inflates self-assessment.
- Illusion of control — overestimates control over outcomes.
- Illusory superiority — rates self above average.
- Valence effect — overestimates probability of good events.
- Hard-easy effect — miscalibrates confidence vs task difficulty.
- Wishful thinking — preference shapes probability estimate.
- Hot-hand fallacy — expects streaks to continue.

## 3. Social / Conformity
- Groupthink — consensus pressure suppresses dissent.
- Bandwagon effect — adopts beliefs as adoption spreads.
- Authority bias — over-weights authority opinion.
- Halo effect — one trait colors overall judgment.
- In-group bias — favors one's own group.
- Social proof — copies others under uncertainty.
- False-consensus effect — overestimates how many agree.
- Conformity — aligns to perceived group norm.
- Courtesy bias — withholds true opinion to avoid offense.
- Herd behavior — follows the crowd over private signal.

## 4. Memory / Availability
- Availability heuristic — weights easily recalled instances.
- Recency bias — over-weights latest information.
- Salience bias — focuses on prominent features.
- Survivorship bias — analyzes survivors, ignores failures.
- Hindsight bias — "knew it all along" after the fact.
- Peak-end rule — judges experience by peak and end.
- Frequency illusion — recent notice inflates perceived frequency.
- Primacy effect — over-weights first information.
- Von Restorff effect — distinctive item over-remembered.
- Rosy retrospection — past rated more positively than experienced.

## 5. Framing / Anchoring
- Anchoring — initial value dominates estimate.
- Framing effect — presentation changes the decision.
- Contrast effect — judgment shifted by comparison.
- Decoy effect — third option steers choice.
- Money illusion — nominal over real value.
- Denomination effect — spends small units more readily.
- Distinction bias — over-differentiates when comparing jointly.
- Unit bias — treats a unit as the right amount.
- Default effect — accepts the pre-set option.
- Focusing effect — over-weights one attribute.

## 6. Attribution
- Fundamental attribution error — over-attributes others to character.
- Self-serving bias — credit for success, blame externalized.
- Actor-observer bias — self situational, others dispositional.
- Just-world hypothesis — assumes outcomes are deserved.
- Defensive attribution — blame shifts with perceived similarity.
- Group-attribution error — individual seen as group-representative.
- Trait-ascription bias — sees self as variable, others fixed.
- Illusion of transparency — overestimates how readable one is.
- Naive realism — believes one sees reality objectively.
- Illusion of asymmetric insight — knows others better than they know self.

## 7. Decision / Loss
- Loss aversion — losses weighted more than equal gains.
- Sunk-cost fallacy — continues due to past investment.
- Status-quo bias — prefers current state.
- Endowment effect — overvalues what is owned.
- Escalation of commitment — doubles down on failing course.
- Ambiguity aversion — avoids unknown probabilities.
- Zero-risk bias — over-pays to eliminate small risk fully.
- Pseudocertainty effect — treats probable as certain when framed so.
- Disposition effect — sells winners, holds losers.
- Omission bias — judges harm by action worse than by inaction.

## 8. Statistical / Probabilistic
- Base-rate neglect — ignores prior probabilities.
- Gambler's fallacy — expects reversal after a run.
- Conjunction fallacy — rates conjunction above its parts.
- Regression-to-mean neglect — treats normal regression as effect.
- Sample-size insensitivity — ignores small-sample volatility.
- Clustering illusion — sees patterns in randomness.
- Ludic fallacy — applies clean models to messy reality.
- Neglect of probability — ignores probability under emotion.
- Denominator neglect — focuses on numerator, ignores base.
- Texas-sharpshooter fallacy — selects data cluster post hoc.
